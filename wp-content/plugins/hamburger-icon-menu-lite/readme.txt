This plugin is allows you to create side or fullscreen menu with hamburger icon. You can set different options such as menu bar and icon color etc.

== Description ==

Are you looking for a beautifully simple, yet fully functional, Hamburger Icon Fly-out Menu for WordPress?

**Look no further!**

This easy to install WordPress plugin will deliver any format of menu you can think of using the popular Hamburger Icon!

**Features included**:

* Five menu layouts
* Colour Options
* Size Options
* Google Font Support
* Side flyout with Hamburger Icon
* Fullscreen menu with Hamburger Icon

**No Coding Required!**

All options actioned from inside your existing WordPress Admin Dashboard.

**You have full control of different parameters including:**

* Menu Bar
* Icon Color
* Size
* Position
* Menu Item Animations

Need more Features and Options? [Got it](http://codecanyon.net/item/hamburger-icon-menu-wordpress-plugin/11620163?ref=borisolhor)

== Installation ==

There is several ways to install this plugin:

1. Install via WordPress Plugins manager panel (Plugins -> Add New -> Search -> Install Now ).
2. Download zip archive from wordpress.org and install via WordPress Plugins manager (Plugins -> Add New -> Upload Plugin -> Install Now).
3. Download zip archive from wordpress.org, unpack it and upload `hamburger_icon_menu_lite` folder to `wp-content/plugins/` on your site via FTP.
