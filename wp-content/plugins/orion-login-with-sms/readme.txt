This plugin allows you to Login with OTP/SMS Verification Code with supported forms.

== Description ==

It lets your user login on to your website simply with their mobile number. No more password or email required.

== Features ==

1. **Passwordless login with SMS/OTP**
2. Compatible with All Themes and Plugins
3. **Fast and Secure**
4. **Uses React :** Lightening fast experience for users.
5. **Responsive :** Works seamlessly for all devices
6. **Translation Ready:** Compatible with WPML and RTL
7. Non Developer Friendly
8. Free Updates
9. **Free Single International Country:** Use it in your country.

== Installation and Use ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WordPress admin Dashboard under 'Orion Login Settings' Menu and fill the required fields.
4. Get the required API key, Secret and Sender IDs etc from one of the above mentioned SMS service providers ( as per the instructions shown in the settings page )
5. Fill the required fields as described in the plugin's settings page.
6. Put the form selector class name or id on the plugin settings page and the plugin will automatically create a 'Login with SMS' button in your login form.
7. When the user clicks on the 'Login with SMS' button a new form will pop up, where user can enter the his phone number that he gave at the time of registration.
8. User can receive the verification code on the registered mobile number. If he verifies the code correctly then he will be logged in and directed to home page.
9. Plugin loads font-awesome( open source, gpl friendly ) from cdn for admin dashboard settings. However when offline everything should work as expected, except the icons would show when you are back online.

== Frequently Asked Questions ==

= Its not working.

Step 1. Check if your Plugin is activated.
Step 2. Deactivate all plugins and reactivate orion-login-with-sms.
Step 3. Check if all required fields are filled in the WordPress admin Dashboard under Orion Login Settings.
